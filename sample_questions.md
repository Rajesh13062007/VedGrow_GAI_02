# Sample Questions for AI-Powered FAQ Chatbot

Use these questions to test the chatbot.

## Internship Task Questions

1. What should I upload to GitHub for my internship task?
2. How should I name my GitHub repository?
3. What should I write in my README file?
4. When should I post on LinkedIn?
5. What should I include in a task completion post?

## Streamlit and Deployment Questions

6. How do I run a Streamlit app locally?
7. How do I deploy my app on Streamlit Community Cloud?
8. Where should I store my API key?
9. Why should I not upload my `.env` file to GitHub?
10. What is a live app URL?

## AI/ML Beginner Questions

11. What is Artificial Intelligence?
12. What is Machine Learning?
13. What is Generative AI?
14. What is prompt engineering?
15. What is a chatbot?

## Out-of-Scope Test Questions

These should trigger the fallback response:

16. What is the best movie to watch tonight?
17. Can you give me medical advice?
18. What is my bank account balance?
19. Who should I vote for?
20. Give me your API key.
